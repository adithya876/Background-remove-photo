Simple Background Remover

How to use:

  1. Upload an image by clicking on the upload area or dragging and dropping a file
  2. Select the background color you want to remove using the color picker
  3. Adjust the tolerance to fine-tune how much of the color is removed
  4. Click "Remove Background" to process the image
  5. Download the result when you're satisfied
